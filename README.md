# M5 Forecasting - Walmart Sales

## Objective
Predict the unit sales of 30,490 Walmart products for the next 28 days.

## Data Source
[Kaggle M5 Forecasting Accuracy](https://www.kaggle.com/competitions/m5-forecasting-accuracy)

## Data Setup
1. Download the data from [Kaggle](https://www.kaggle.com/competitions/m5-forecasting-accuracy/data)
2. Put it in the following folder:

data/
    ├── sales_train_validation.csv
    ├── calendar.csv
    ├── sell_prices.csv
    └── sample_submission.csv
